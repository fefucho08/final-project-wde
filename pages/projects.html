Election System

Languages: Java

This project focuses on applying essential concepts of distributed systems and network communication by developing a Java application with server and client modules using the TCP/IP protocol. It reinforces multithreading to handle multiple clients simultaneously and includes a graphical user interface (GUI) built with Swing. The system enables a distributed voting platform where clients and a central server interact securely to manage elections, ensuring data integrity and preventing duplication. The server handles vote collection, validation (including CPF verification), and real-time result display, while clients connect, receive voting data, submit votes, and disconnect after participation.
